# :iphone: Project Title

### AI 나이측정 테스트, How Old Am I

## Key Features

* HandPose 인식
  - CreateML로 학습된 모델로 Camera에서 손을 인식하고 미리 학습된 손하트 포즈를 인식합니다.
* 3D 세계에 하트 배치
  - 하트가 손하트의 포지션을 따라가고 일정시간 한 위치에 고정된 경우 픽스합니다.
* 갤러리에 저장
  - 3D 개체들을 배치한 후 함께 사진을 찍을 수 있습니다.
